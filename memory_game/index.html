Instructions

Concentration, also known as Match Match, Memory, Pelmanis, Shinkei-suijaki, Pexeso or simply pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards

This game is great for beginners
